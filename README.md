# ‚ú® Gleam - Chrome Extension

Curate your digital wisdom. Select text, share it, and save it with an elegant UI.

## Features

### Core Functionality
- **Highlight & Share**: Select any text on a webpage and click the "‚ú® Share!" tooltip.
- **Elegant Cards**: View your selected text in a beautifully designed, Neumorphic card.
- **Save Snippets**: Save your favorite quotes and highlights for later.
- **Download as Image**: Generate high-quality images of your highlights to share on social media.

### Management
- **Dashboard**: View all your saved highlights in a clean, Neumorphic popup.
- **Stats**: Track your total saved highlights and daily activity.
- **Review**: Re-open the elegant card view for any saved highlight.
- **Manage**: Delete old highlights you no longer need.

## Installation

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the extension folder
5. The Gleam icon will appear in your toolbar

## Usage

### Highlighting
1. Select any text on a webpage.
2. Click the "‚ú® Share!" tooltip that appears.
3. In the modal, choose to **Save** or **Download Image**.

### Managing Highlights
1. Click the **Gleam icon** in your browser toolbar.
2. View your list of saved highlights.
3. Click **View** to see the card again or download the image.
4. Click **Delete** to remove a highlight.

## Data Privacy

All data is stored locally in your Chrome browser using the `chrome.storage.local` API. No data is sent to external servers.
